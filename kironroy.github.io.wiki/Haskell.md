### GHCi in Terminal (basics)

`$ ghci`

`$ GHCi, version 6.8.2: http://www.haskell.org/ghc/  :? for help  
Loading package base ... linking ... done.  
Prelude>`

`:set prompt "ghci> "`

`ghci> 2 + 15`

`17`

`ghci> 49 * 100`

`4900`

`ghci> 1892 - 1472`

`420`

`ghci> 5 / 2 `

`2.5`
  
***

`ghci> True && False`  

`False`  

`ghci> True && True`  

`True`  

`ghci> False || True`  

`True`   

`ghci> not False`  

`True`  

`ghci> not (True && True)`  

`False`  


***

`ghci> 5 == 5`  

`True`  

`ghci> 1 == 0`  

`False`  

`ghci> 5 /= 5`  

`False`  

`ghci> 5 /= 4`  

`True`  

`ghci> "hello" == "hello"`  

`True`
